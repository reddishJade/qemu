main.c中，通过
ret = loader_exec(execfd, exec_path, target_argv, target_environ, regs);
来加载目标平台的程序，然后通过
cpu_loop(env);
来进入模拟CPU循环

循环中调用x86_64/../i386/cpu_loop.c中的
trapnr = cpu_exec(cs);
来执行CPU指令，调用动态翻译获取翻译块(translation block TB)，cs代表代码段寄存器(code segment register)

cpu_exec()调用accel/tcg/cpu-exec.c中的
tb = tb_lookup(cpu, pc, cs_base, flags, cflags);
在TB缓存中查找翻译块，如果已经翻译过并且还在缓存中则直接返回缓存的翻译块
没有找到翻译块则调用tb_gen_code来生成新的翻译块,代码为
tb = tb_gen_code(cpu, pc, cs_base, flags, cflags);

tb_gen_code()调用translate-all.c中的
return tcg_gen_code(tcg_ctx, tb, pc);
tcg_gen_code()使用TCG来执行动态翻译，tcg_ctx：翻译上下文，tb：翻译块，pc：当前的程序计数器

然后到cpu_exec.c中，通过
strlret = tcg_qemu_tb_exec(env, tb_ptr);
来将已经翻译好的机器码交给主机 CPU 执行，
最后重复循环
add tb = tb_lookup(cpu, pc, cs_base, flags, cflags);
add tb = tb_gen_code(cpu, pc, cs_base, flags, cflags);
return tcg_gen_code(tcg_ctx, tb, pc);
查找翻译块和生成翻译块

动态翻译的关键函数：tb_lookup()和tb_gen_code()
目标平台代码存放在模拟的内存中，通过程序计数器（PC）读取
翻译后的代码（主机平台代码）：存放在翻译块缓存（Translation Cache）中，供主机平台执行。
